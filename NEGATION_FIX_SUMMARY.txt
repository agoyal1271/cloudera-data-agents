HOW IT WORKS
================================================================================

Query: "find tables NOT in ozone"

Step 1: EMBED GOAL
  goal_vector = embed("find tables NOT in ozone")
  → [0.23, 0.87, 0.12, ...]

Step 2: SEARCH INTENT CACHE
  results = qdrant.search(query_vector=goal_vector, threshold=0.92)
  → Found similar goal with similarity=0.94 (cache hit!)
  → Return stored intent

Step 3: STORED INTENT
  {
    asset_types: ["iceberg_table"],
    storage: {value: "ozone", negate: true}
  }

Step 4: CONVERT TO FILTERS
  metadata_filters = {"storage": "!ozone"}

Step 5: APPLY FILTERS
  For each asset:
    if filters["storage"].startswith("!"):
      if asset.storage == "ozone": reject
    else:
      if asset.storage != "ozone": reject

Step 6: RETURN RESULTS
  Assets in s3a, hdfs, azure (NOT ozone) ✓
